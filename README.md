# CYBER_TRANS // LOCAL

A futuristic, completely local multi-language translation dashboard powered by Flask and `deep-translator`.

## 🚀 Features

- **Cyberpunk UI**: A state-of-the-art neon aesthetic with glitch headers, custom form controls, and dynamic hover effects.
- **Local Translations**: Translate English text into multiple global languages (Spanish, French, German, Chinese, Japanese, etc.) entirely for free without needing an Anthropic API key.
- **Custom Aesthetic**: Built using 100% Vanilla CSS to achieve intricate glowing shadows, geometric clip-paths, scanlines, and vignette effects.
- **Asynchronous Processing**: Fetches and renders translations smoothly without reloading the page using the JavaScript Fetch API.

## 🛠️ Technology Stack

- **Backend**: Python 3, Flask
- **Translation Engine**: `deep-translator` (Google Translate free endpoint)
- **Frontend**: HTML5, Vanilla CSS3, Vanilla JavaScript
- **Fonts**: Orbitron, Rajdhani (Google Fonts)

## 📥 Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BHUVAN2525/Translator.git
   cd Translator
   ```

2. **Install dependencies:**
   Ensure you have Python installed. Then run:
   ```bash
   pip install flask deep-translator
   ```

3. **Initialize the Server:**
   ```bash
   python app.py
   ```

4. **Access the Interface:**
   Open your browser and navigate to the local server address: `http://127.0.0.1:5000/`.
